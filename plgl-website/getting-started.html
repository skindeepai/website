Getting Started with PLGL

Build personalized AI applications using preference learning and generative models

Introduction

PLGL (Preference Learning in Generative Latent Spaces) enables you to create AI applications that learn and adapt to individual user preferences. This guide will walk you through everything you need to start building with PLGL.

What You'll Learn

  • How to set up PLGL in your project
  • Understanding the core concepts
  • Building your first preference-learning application
  • Best practices and optimization techniques

Installation

Using pip

pip install plgl

# For specific deep learning frameworks
pip install plgl[torch]     # PyTorch support
pip install plgl[tf]        # TensorFlow support
pip install plgl[jax]       # JAX support

From Source

git clone https://github.com/plgl/core.git
cd core
pip install -e .

Requirements

  • Python 3.8+
  • A deep learning framework (PyTorch, TensorFlow, or JAX)
  • A pre-trained generative model with accessible latent space

Quick Start

Here's a minimal example to get you started with PLGL:

from plgl import PreferenceLearner
from your_model import load_generator  # Your generative model

# 1. Load your generative model
generator = load_generator('path/to/model')

# 2. Initialize PLGL
learner = PreferenceLearner(
    generator=generator,
    latent_dim=512,  # Your model's latent dimension
    device='cuda'
)

# 3. Collect user preferences
print("Rate the generated samples (1-5):")
preferences = learner.collect_preferences(n_samples=20)

# 4. Train preference model
learner.train(preferences)

# 5. Generate personalized content
optimal_content = learner.generate_optimal()
print("Generated personalized content!")

Core Concepts

1. Generative Models & Latent Spaces

PLGL works with any generative model that has a continuous latent space. This includes:

  • GANs (StyleGAN, ProGAN, etc.)
  • VAEs (Variational Autoencoders)
  • Diffusion Models with latent representations
  • Any model where inputs map to outputs through a continuous space

2. Preference Learning

PLGL learns preferences through various rating mechanisms:

# Binary preferences (like/dislike)
rating = learner.get_binary_rating(content)  # True/False

# Scaled preferences (1-5 stars)
rating = learner.get_scaled_rating(content)  # 1-5

# Comparative preferences (A vs B)
preference = learner.get_comparison(content_a, content_b)  # 0 or 1

# Multi-criteria preferences
ratings = learner.get_multicriteria_rating(content, 
    criteria=['style', 'quality', 'relevance']
)

3. Latent Space Navigation

PLGL optimizes through the latent space to find content matching learned preferences:

# Gradient-based optimization
optimal_z = learner.optimize_latent(
    initial_z=torch.randn(1, latent_dim),
    steps=1000,
    learning_rate=0.01
)

# Evolutionary optimization (for non-differentiable models)
optimal_z = learner.optimize_evolutionary(
    population_size=50,
    generations=100
)

# Distribution generation
distribution = learner.generate_distribution(
    n_samples=20,
    diversity=0.5  # Balance between quality and diversity
)

Your First PLGL Project

Let's build a complete preference-learning application step by step.

Project: Personalized Art Generator

Step 1: Setup

import torch
from plgl import PreferenceLearner, RatingInterface
from torchvision.utils import save_image
import matplotlib.pyplot as plt

# For this example, we'll use a pre-trained StyleGAN2
from stylegan2_pytorch import StyleGAN2

# Load model
generator = StyleGAN2.load('stylegan2-ffhq-config-f.pt')
latent_dim = 512

Step 2: Create Rating Interface

class ArtRatingInterface(RatingInterface):
    """Custom interface for rating artwork"""
    
    def display_content(self, content):
        """Display generated artwork"""
        plt.figure(figsize=(8, 8))
        plt.imshow(content.permute(1, 2, 0).cpu())
        plt.axis('off')
        plt.show()
    
    def get_rating(self, content):
        """Get user rating"""
        self.display_content(content)
        
        while True:
            rating = input("Rate this artwork (1-5, or 's' to skip): ")
            if rating == 's':
                return None
            try:
                rating = int(rating)
                if 1 <= rating <= 5:
                    return rating / 5.0  # Normalize to [0, 1]
            except ValueError:
                pass
            print("Invalid input. Please enter 1-5 or 's'.")

# Initialize learner with custom interface
learner = PreferenceLearner(
    generator=generator,
    latent_dim=latent_dim,
    rating_interface=ArtRatingInterface()
)

Step 3: Collect Preferences

# Smart sampling for diverse initial examples
from plgl.sampling import DiversitySampler

sampler = DiversitySampler(latent_dim=latent_dim)
initial_samples = sampler.sample(n=30, diversity=0.8)

# Collect ratings
print("Please rate the following artworks:")
preferences = learner.collect_preferences(
    initial_latents=initial_samples,
    min_ratings=20  # Minimum ratings before training
)

Step 4: Train Preference Model

# Train with validation
from plgl.models import DeepPreferenceModel

preference_model = DeepPreferenceModel(
    input_dim=generator.output_dim,
    hidden_dims=[512, 256, 128],
    dropout=0.2
)

history = learner.train(
    preferences,
    model=preference_model,
    epochs=200,
    validation_split=0.2,
    early_stopping=True
)

# Visualize training
plt.plot(history['loss'], label='Training Loss')
plt.plot(history['val_loss'], label='Validation Loss')
plt.xlabel('Epoch')
plt.ylabel('Loss')
plt.legend()
plt.show()

Step 5: Generate Personalized Art

# Generate single optimal artwork
optimal_art = learner.generate_optimal(
    optimization_steps=2000,
    learning_rate=0.005
)

save_image(optimal_art, 'personalized_artwork.png')
print("Saved your personalized artwork!")

# Generate a collection
collection = learner.generate_collection(
    n_pieces=10,
    diversity=0.3,  # Some variety while maintaining preferences
    min_score=0.8   # Quality threshold
)

for i, art in enumerate(collection):
    save_image(art, f'collection/artwork_{i}.png')

Step 6: Active Learning (Optional)

# Improve preferences with active learning
from plgl.active import UncertaintySampler

active_sampler = UncertaintySampler(learner.preference_model)

for round in range(3):
    print(f"\nActive Learning Round {round + 1}")
    
    # Get most informative samples
    uncertain_samples = active_sampler.select_samples(
        n_samples=10,
        method='entropy'
    )
    
    # Collect ratings for these specific samples
    new_preferences = learner.collect_preferences(
        initial_latents=uncertain_samples
    )
    
    # Retrain with all data
    learner.update_preferences(new_preferences)
    learner.train(learner.all_preferences)

Advanced Usage

Multi-Modal Preferences

from plgl.multimodal import MultiModalLearner

# Combine visual and textual preferences
learner = MultiModalLearner(
    generators={
        'visual': visual_generator,
        'text': text_generator
    },
    fusion_method='attention'
)

# Learn from multiple modalities
preferences = learner.collect_preferences(
    modalities=['visual', 'text'],
    n_samples_per_modality=20
)

Conditional Generation

# Generate content with conditions
from plgl.conditional import ConditionalPLGL

conditional_learner = ConditionalPLGL(generator, latent_dim)

# Learn preferences for different contexts
preferences_happy = learner.collect_preferences(condition='happy')
preferences_calm = learner.collect_preferences(condition='calm')

# Generate based on mood
happy_content = conditional_learner.generate(
    condition='happy',
    user_preferences=preferences_happy
)

Real-Time Adaptation

from plgl.realtime import AdaptiveLearner

# Continuously adapt to user feedback
adaptive = AdaptiveLearner(
    generator,
    learning_rate=0.001,
    buffer_size=100
)

# In your application loop
while True:
    content = adaptive.generate()
    display(content)
    
    feedback = get_user_feedback()  # Implicit or explicit
    adaptive.update(content, feedback)
    
    # Model continuously improves
    if adaptive.confidence > 0.9:
        print("Model has high confidence in preferences!")

Distributed Learning

from plgl.distributed import FederatedPLGL

# Learn from multiple users while preserving privacy
federated = FederatedPLGL(
    generator,
    aggregation='fedavg',
    differential_privacy=True
)

# Each user trains locally
local_model = federated.create_local_model(user_id='user123')
local_model.train(user_preferences)

# Aggregate learnings
global_model = federated.aggregate_models(
    local_models=[model1, model2, model3],
    weights=[0.3, 0.4, 0.3]
)

Troubleshooting

Common Issues

Preference model not converging

Solution: Try these approaches:

# 1. Collect more diverse preferences
sampler = DiversitySampler(method='maxmin')
diverse_samples = sampler.sample(n=50)

# 2. Use a simpler model first
simple_model = LinearPreferenceModel()

# 3. Adjust learning rate
learner.train(preferences, learning_rate=0.0001)

# 4. Check for contradictory preferences
stats = learner.analyze_preferences()
print(f"Consistency score: {stats['consistency']}")

Generated content lacks diversity

Solution: Increase exploration:

# Use temperature sampling
samples = learner.generate_distribution(
    n_samples=20,
    temperature=1.5,  # Higher = more diversity
    min_distance=0.3  # Ensure variety
)

# Or use different optimization starting points
starts = torch.randn(10, latent_dim) * 2  # Wider distribution
results = [learner.optimize_from(z) for z in starts]

Memory issues with large models

Solution: Use efficient techniques:

# Gradient checkpointing
learner = PreferenceLearner(
    generator,
    gradient_checkpointing=True
)

# Batch processing
for batch in learner.process_in_batches(preferences, batch_size=4):
    process(batch)

# Use mixed precision
with torch.cuda.amp.autocast():
    result = learner.generate_optimal()
